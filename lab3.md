** Differences between the TypeScript and JavaScript
** Javascript:
-> It used for small projects
-> It is Dynamically typed
-> It works on Runtime represents
-> Standard JavaScript syntax
-> Limited built-in tooling
 

 ** Typescript
-> It used for large projects
-> It is Statically typed
-> It works on Compile represents
-> Similar to JavaScript, with additional features 
-> Comes with IDEs and code editors
